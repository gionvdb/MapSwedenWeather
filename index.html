Interactive Map of Sweden

Click on one of the places to hear the sonification of 
the average temperature over the last 20 years!
The data was retrieved from the SMHI, 
then processed to .mid files using python with MIDITime library. 
The sounds were processed using FL Studio 
and the TAL U-NO-LX plugin.
The interactive map was made using HTML, CSS 
and the WebAudioXML library.

All sources will be linked to below. 
Meteorological Data of Sweden by SMHI. 
https://www.smhi.se/data/meteorologi
MIDITime by Reveal and Micheal Corey. 
https://github.com/cirlabs/miditime
WebAudioXML by Hans Lindetorp. 
https://github.com/hanslindetorp/WebAudioXML
Image Map of Sweden by Vemaps. 
https://vemaps.com/sweden/se-01
